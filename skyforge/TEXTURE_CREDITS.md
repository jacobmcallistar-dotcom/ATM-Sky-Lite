# SkyForge texture credits

All SkyForge item/block textures were previously placeholder reuses of vanilla Minecraft
textures. As of v0.15.0 they use original art, sourced only from explicitly CC0
(public domain) works — no attribution is legally required, but it's listed here for
transparency and easy re-verification.

| SkyForge asset | Source | License |
|---|---|---|
| `item/sky_dust.png` | Generated by Claude for this project (procedural, original) | N/A — original work |
| `item/sky_steel_ingot.png` | `default_steel_ingot.png`, [Assorted Minecraft style textures](https://opengameart.org/content/assorted-minecraft-style-textures) (derived from the Minetest `default` game assets) | CC0 |
| `item/industrial_keystone.png` | `default_mese_crystal.png`, [Assorted Minecraft style textures](https://opengameart.org/content/assorted-minecraft-style-textures) | CC0 |
| `item/heart_of_the_sky.png` | `terminus.png`, [CC0 Minecraft Inspired Textures](https://opengameart.org/content/cc0-minecraft-inspired-textures) | CC0 |
| `block/sky_assembler_top.png` | Composite: `default_steel_block.png` ([Assorted Minecraft style textures](https://opengameart.org/content/assorted-minecraft-style-textures), CC0) + a gear from [Cogwheels and Gears](https://opengameart.org/content/cogwheels-and-gears) (CC0) | CC0 |
| `block/sky_assembler_side.png` | Same composite sources as above | CC0 |
| `block/sky_assembler_bottom.png` | `default_steel_block.png`, [Assorted Minecraft style textures](https://opengameart.org/content/assorted-minecraft-style-textures) | CC0 |
